Section 1 : Les balises de texte

Titre : Les balises HTML de base pour le texte

En HTML, la balise p sert à créer un paragraphe. Les titres vont de h1 à h6. Pour mettre en valeur un mot, on utilise strong (gras) ou em (italique). Ces balises sont purement sémantiques.

Section 2 : Fontes et polices

Titre : La propriété font-family

font-family permet de spécifier la police utilisée pour un texte. Il est recommandé d’indiquer plusieurs polices, suivies d’une famille générique (ex. : "Roboto", Arial, sans-serif).

Section 3 : Taille, poids, style

Titre : Contrôler l’apparence du texte

font-size contrôle la taille du texte. font-weight définit l’épaisseur (normal, bold, 100–900). font-style permet de passer en italique.

Section 4 : Espacement et alignement

Titre : Améliorer la lisibilité

line-height gère l’espace entre les lignes. letter-spacing ajuste l’espace entre les lettres. text-align permet d’aligner le texte à gauche, à droite, centré ou justifié.

Section 5 : Couleur et contraste

Titre : Styliser avec color

La propriété color permet de changer la couleur du texte. Il est essentiel de choisir des couleurs lisibles sur le fond de la page.
